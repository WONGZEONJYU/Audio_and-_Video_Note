# 1. FFmpeg filter简介

FFmpeg filter提供了很多⾳视频特效处理的功能 

⽐如视频缩放、截取、翻转、叠加等 , 其中定义了很多的filter , 例如以下常⽤的⼀些filter

- scale : 视频/图像的缩放
- overlay : 视频/图像的叠加
- crop : 视频/图像的裁剪
- trim : 截取视频的⽚段
- rotate : 以任意⻆度旋转视频

⽀持的filter的列表可以通过以下命令获得

> ```bash
> ffmpeg -filters
> ```



也可以查看⽂档[2]，具体某个版本的⽀持情况以命令⾏获取到的结果为准。
以下是filter的⼀个简单的应⽤示例 , 对视频的宽和⾼减半

> ```bash
> ffmpeg -i input -vf scale=iw/2:ih/2 output
> ```
